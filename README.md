# nwprinter-kit
大学で配布されているVBS ScriptがWindows Defenderに弾かれるのでC#で書いたやつ。

### 使い方。
- [releasesページ](https://github.com/nikaidoumari/nwprinter-kit/releases) から最新版をダウンロード
- 実行する。
- 認証する。
- プリンターに繋がってワイワイ

### バグ報告・機能追加要望
[issues](https://github.com/nikaidoumari/nwprinter-kit/issues)から報告してください。
